# Portadas de cursos

Coloca aquí las imágenes de portada de cada curso.

**Archivos esperados:**
- `acompanamiento-social.jpg`
- `primeros-auxilios-emocionales.jpg`
- `comunicacion-empatica.jpg`

**Recomendaciones:**
- Formato: JPG o WebP.
- Ratio: 16:9 horizontal.
- Resolución mínima: 1280×720 px.

El nombre del archivo debe coincidir con el `id` del curso en `src/lib/mock.ts`.
